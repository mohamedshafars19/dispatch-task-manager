# 🚉 Dispatch Task Manager

A modern task management web application inspired by **vintage railway departure boards**, where every task is displayed like a departure ticket and the next upcoming deadline is highlighted through a live countdown display. Designed to combine productivity with a unique visual experience.

## ✨ Features

* 🎫 Railway departure board inspired interface
* ⏳ Live countdown timer for the next upcoming task
* 📝 Create tasks with custom due date and time
* 🚦 Priority management (Low, Medium, High)
* 🔔 Browser notification reminders
* ⏰ Built-in reminder scheduling
* ⚠️ Automatic overdue task detection
* ✅ Mark tasks as completed
* 🔍 Filter tasks based on status and priority
* 📱 Fully responsive design
* ⚡ Built with pure HTML, CSS & JavaScript
