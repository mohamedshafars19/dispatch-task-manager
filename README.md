# dispatch-task-manager

A unique task management web application designed around the concept of a vintage railway departure board. Every task becomes a ticket, and the next upcoming deadline is displayed on a live countdown board.

## Features

- Live countdown timer for the next task
- Vintage departure board inspired UI
- Task creation with due date and time
- Priority levels (Low, Medium, High)
- Desktop notification reminders
- Built-in reminder scheduling
- Overdue task detection
- Mark tasks as completed
- Task filtering
- Responsive design
- Pure HTML, CSS and JavaScript

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
